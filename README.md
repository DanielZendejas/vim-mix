# mix.vim

Plugin for using Elixir's build tool, `mix`.

## Installation

I recommend installing [pathogen.vim][pathogen], then running the following:

    cd ~/.vim/bundle
    git clone git://github.com/mattonrails/vim-mix.git

## Usage

- `:Mixdo`
